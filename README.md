## Project Overview
FoodCorner is a full-stack food ordering web application where users can browse food items, add items to the cart, and place orders online.

Orders are stored in MongoDB Atlas through a Node.js and Express backend API.

## Features
- Browse food menu
- Add food items to cart
- Calculate total order amount
- Place orders
- Store orders in MongoDB Atlas
- Responsive UI
- Full-stack deployment

## Installation

### Clone Repository

```bash
git clone https://github.com/subhashSRKR24/FoodCorner-FullStack.git
```

### Install Dependencies

```bash
npm install
```

### Run Backend

```bash
node server.js
```

## Database Setup

1. Create MongoDB Atlas Cluster
2. Create Database User
3. Add IP Access List
4. Configure MongoDB Connection String
